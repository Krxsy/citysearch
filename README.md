# citysearch

Usage
------
Initialize the server

```
$ make start
```

open your browser and type **yourserver:9999/search.html**.

Request a city which will be shown on google maps.


Dataset
-------
The city2 dataset is gained from the Chair for Algorithms and Data Structures, Department of Computer Science, University of Freiburg, headed by Hannah Bast.

Please note
-----------
This is just a primitive search engine developed for fun and learning purposes.
